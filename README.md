# Aero32Alternative
Resized windows default cursors for 1080p or smaller screens at 100 DPI. 

`Hello fellow internet dweller, do you also lowkey have OCD? 
did you also always find the default windows 10 cursor at 100 dpi to be very ugly but thought the large scheme was "ok-looking"?
did you also refrain from using custom cursors because most went against the default windows design language which further triggered your OCD? 
Then I've got your back my friend, behold! the windows 10 large cursor scheme but resized for 100 dpi 1080p or 766p screens!`
